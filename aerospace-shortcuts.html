AeroSpace Shortcuts

Window Focus (⌥ + h/j/k/l)

⌥ h
Focus left
⌥ j
Focus down
⌥ k
Focus up
⌥ l
Focus right

Move Window (⇧⌥ + h/j/k/l)

⇧⌥ h
Move left
⇧⌥ j
Move down
⇧⌥ k
Move up
⇧⌥ l
Move right

Resize Window (⌃⌥ + h/j/k/l)

⌃⌥ h
Decrease width
⌃⌥ j
Increase height
⌃⌥ k
Decrease height
⌃⌥ l
Increase width

Workspaces

⌥ 1-5
Switch to workspace 1-5
⇧⌥ 1-5
Move window to workspace 1-5
1 iTerm
2 Brave
3 VS Code
4 Claude
5 Notes/Finder/etc

Monitors

⌥ i
Focus left monitor
⌥ o
Focus right monitor
⇧⌥ i
Move window to left monitor
⇧⌥ o
Move window to right monitor

Window Management

⌥ f
Toggle fullscreen
⌥ t
Toggle float/tile
⌥ r
Rotate layout (H/V)
⌥ b
Balance window sizes
⌥ ↩
Launch iTerm

⌥ = Option/Alt | ⇧ = Shift | ⌃ = Control | ↩ = Enter
